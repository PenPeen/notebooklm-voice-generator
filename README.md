# NotebookLM Auto Generator

クリップボードにあるURLを使用して、NotebookLMで自動的に新しいノートブックを作成し、音声要約の生成を開始するChrome拡張機能です。

## 機能
- クリップボードのURLを自動読み取り
- NotebookLMの新規作成、ソース追加、音声生成フローを自動化

## インストール手順

1. このリポジトリをダウンロードまたはクローンします。
2. Google Chromeを開き、アドレスバーに `chrome://extensions` と入力します。
3. 右上の「デベロッパーモード」スイッチをオンにします。
4. 「パッケージ化されていない拡張機能を読み込む」ボタンをクリックします。
5. このプロジェクトのフォルダ (`notebooklm-automation`) を選択します。

## 使用方法

1. 読み込ませたいウェブサイトのURLをコピーします (クリップボードに保存)。
2. Chromeのツールバーにある拡張機能アイコンをクリックします。
3. 「クリップボードのURLで作成開始」ボタンをクリックします。
4. NotebookLMが新しいタブで開き、自動操作が開始されます。
   - **注意**: 自動操作中はマウスやキーボード操作を行わないでください。

## トラブルシューティング

- **要素が見つからないエラー**: NotebookLMのUIが更新された可能性があります。`content.js` のセレクタを修正する必要があります。
- **動作しない場合**: 拡張機能のアイコンを右クリックし、「検証」を選択してコンソールログ (`Console`) を確認してください。

## 開発者向け

- `content.js`: DOM操作のメインロジック。`DEBUG = true` に設定されています。
- `background.js`: タブの作成とメッセージング制御。# notebooklm-voice-generator
